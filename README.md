# 🛒 MERN E-Commerce Platform

A full-featured e-commerce application built with the MERN stack (MongoDB, Express.js, React, Node.js).

## 🚀 Features

- **User Authentication**: JWT-based authentication with signup/login
- **Product Management**: Browse products with categories and search
- **Shopping Cart**: Add/remove items, update quantities
- **Order Processing**: Place orders with address management
- **Address Book**: Save multiple addresses with default selection
- **Admin Panel**: Manage products, orders, and users
- **Payment Integration**: Ready for Stripe/PayPal integration
- **Responsive Design**: Mobile-friendly with Tailwind CSS
- **State Management**: Redux Toolkit for global state
- **Security**: Password hashing, protected routes, input validation

## 🛠️ Tech Stack

### Frontend
- React 18
- Redux Toolkit
- Tailwind CSS
- React Router DOM
- Axios

### Backend
- Node.js
- Express.js
- MongoDB with Mongoose
- JWT Authentication
- Bcrypt.js
